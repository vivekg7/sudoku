Offline Sudoku with smart hints that teach you to think, not just solve.

## Full Description

No ads. No tracking. No internet. Just Sudoku.

A Sudoku app built by a Sudoku nerd, for Sudoku nerds. Every puzzle is generated on the fly, graded by the solving strategies it requires — not just the number of empty cells. From beginner to master, you'll always get a fair challenge.

Smart Hints That Teach You
Most apps just hand you the answer. This one doesn't. Hints are layered — start with a gentle nudge ("look for 7 in box 3"), then a strategy suggestion ("try X-Wing on rows 2 and 7"), and only reveal the full answer if you really want it. Each layer requires a deliberate tap. Your effort is respected.

Every Strategy, Implemented
The solving engine covers all known Sudoku techniques:

- Naked & Hidden Singles, Pairs, Triples, Quads
- Pointing Pairs & Box/Line Reduction
- X-Wing, Swordfish, Jellyfish
- XY-Wing, XYZ-Wing
- Simple Colouring, X-Chains, XY-Chains
- Unique Rectangles, ALS, Forcing Chains & more

6 Difficulty Levels
Beginner → Easy → Medium → Hard → Expert → Master
Each level is defined by which techniques you'll need — so "Hard" actually means hard.

PDF Export
Generate puzzles in bulk and export as PDF for printing. Includes optional rough-work grids, solve-order hints at the back, and QR codes to open any puzzle in the app.

Track Your Growth
Detailed stats track solve times, streaks, completion rates, and exactly which hints you used — so you can see which strategies you're mastering and where you need practice.

Full-Featured Gameplay

- Pencil marks & candidate notes
- Undo / Redo
- Number-first input — tap a digit, then tap cells
- Save & resume, bookmarks
- Export & import all your data as JSON

Open Source
The entire source code is public. No secrets, no hidden tracking. GPLv3 licensed.
